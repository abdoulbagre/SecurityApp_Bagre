# 🔒 SecurityApp_Bagre

Application Android de sécurité qui empêche l’extinction non autorisée de l’appareil.  
➡️ Avant toute tentative d’extinction, l’utilisateur doit entrer un **mot de passe correct**, sinon une **alarme sonore** se déclenche.  

Voir la documentation dans ce fichier pour installation et utilisation.
